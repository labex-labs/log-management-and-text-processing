ログ管理とテキスト処理について学びます。このモジュールでは、標準ログの場所、ログのリアルタイム監視、grep を使用したログの検索、awk を使用した列の処理、sed を使用したストリーム編集を扱います。実践的なラボと実世界の課題を通じて、これらの不可欠な Linux スキルを習得します。
